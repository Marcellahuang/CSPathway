Colleges that offer Web Development Degrees:

1) Seattle central college: Location: Seattle, WA course description and outcomes; Seattle Central College’s Web Development program will provide students with specialized skills needed to build websites including current programming languages, content management systems, databases and web design. program courses: https://it.seattlecentral.edu/programs/web-development/course-planning degree duration: 7 quarters (2 calendar years).

2) Seattle University: Location: Seattle, WA program duration: 4 quarters(1 calendar year). program courses: 1st quarter -WATS Foundations of the Web -WATS Intro to Web Development 2nd quarter -WATS Introduction to Javascript Programming for the Web -WATS Introduction to Servers and Hosting 3rd quarter -WATS Building Javascript Applications for the Web -WATS User Science and Testing 4th quarter -WATS Professional Practice -WATS Web Development Capstone Project

Course outcomes and Job description: -Design Fluency: Learn how to make websites that look good and attract users with tools commonly used by web designers and developers. -Foundational Understanding of the Web: Move from web consumer to web creator with a clear knowledge of how the World Wide Web works, how the Internet works, and how we can use these technologies to create engaging software. -Frontend Web Development:Learn the core fundamentals of HTML, CSS, and JavaScript–the three technologies that make the web
